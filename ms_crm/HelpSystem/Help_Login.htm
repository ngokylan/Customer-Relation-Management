Login form

The Login form allows the Agent to connect to CRM and ISYS application and CRM database.

The Agent has to provide valid authorization a privileges to Milestone Search to get access to CRM and ISYS system by typing correct Username and Password.
It is allowed 3 attempts to type username and password correctly, after that the user is considered suspect and he/she will be locked out from the system, and to reset the account the Administrator has to be contacted.



Username & Password

As usual username and password fields are case sensitive, so please be awarded that password typed in upper-case letters is different than typed in lower-case.

Submit & Clear Button

Clicking the "Submit" button, the Login form will be submitted to the server and after successfully validate it, the Advance Search page (Home Page) of the CRM will be loaded, on the other hand clicking the "Clear" button all the fields of the Login form will be reset.
