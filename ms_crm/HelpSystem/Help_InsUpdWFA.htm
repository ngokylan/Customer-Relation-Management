Insert New / Update Work Flow Action form

A Work Flow Action form is used to assign an automated task with a set duration to a Candidate to aid in the processing of Candidates.
The top section of the form displays candidates details such as candidateID, and the bottom section displays the WFA details assign to the candidate.



How to assign a Work Flow Action to a candidate

The process to assign a WFA to a candidate it is easy:
  • Select the Insert New / Update Work Flow Action bar if you have not already clicked.
  • The field will be auto filled the candidate ID.
  • Specify the WFAID, the form will retrieve automatically a brief description of the WFAID
  • Specify the Assign Template, the form will retrieve automatically the content email of the WFA template associated
  • The status has to be "Active"
  • Fill all the candidate's WFA details in the form.
  • Click the "Save" button.

If the validation will not return any error, the candidate's WFA details will be inserted in the CRM database




How to change a Work Flow Action assigned to a candidate

The process to update a candidate's WFA details is:

  • Select the Insert New / Update Work Flow Action bar if you have not already clicked.
  • All the data will be retrieved from the CRM database.
  • Change the candidate's WFA details in the form.
  • Click the "Update" button.

If the validation will not return any error, the candidate's WFA details will be updated in the CRM database

Submit & Reset Button

Clicking the "Save"/"Update" button the Insert New / Update Candidate form will be submitted to the server, on the other hand clicking the "Reset" button all the fields of the Insert New / Update Candidate form will be re-set.
