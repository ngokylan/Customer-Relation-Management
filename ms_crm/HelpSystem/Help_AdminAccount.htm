Agent Admin Management form

The Agent Admin Management form allow Agent Admin to insert new value in the admin tables such as "Agent", "Client", "Help_System", "Wfa_code", "Wfa_template" and "status" .
For each table has been developed different forms:

Agent Admin Management - Agent form


How to store information about Agents

The process to store information about Agent it is easy:

  • Get access to Milestone Search - CRM blueSky using an Agent Admin user.
  • Click to the tab "Agent".
  • Fill the form.
  • Click the "Save" button.

If the validation will not return any error, the agent user will be create in the CRM database.
