Insert New / Update Candidate details

The Insert New / Update Candidate form is used to add a Candidate into the CRM database or to update an existing candidate's detail.
The top section of the form requires personal candidate's details such as First and Last name, candidate's personal contact details and candidate's working place contact details, then the bottom section requires infomation regardless the candidate's working availability, type of engagement and rates.




How to insert a new candidate

The process to insert a new candidate into the CRM database it is easy:

  • Select the "Insert Candidate" section if you have not already clicked.
  • Fill all the candidate's details in the form.
  • Click the "Save" button.

If the validation will not return any error, the candidate's details will be inserted in the CRM database.




How to update candidate details

The process to update a candidate's details is:

  • Run a search for "Candidate Name" or "Advance Search" to retrieve the details of the researching Candidate
  • Click on the Candidate details and the "Update Candidate" section will be opened automatically, display the Candidate Id.
  • All the data of the Candidate will be retrieved from the CRM database.
  • Change the candidate's details in the form.
  • Click the "Save" button.
If the validation will not return any error, the candidate's details will be changed in the CRM database




Submit & Reset Button

Clicking the "Save" button the "Insert Candidate" form will be submitted to the server which will insert the information, while if the Candidate is already exist in the database the "Update Candidate" from will update Candidate details, on the other hand clicking the "Reset" button all the fields of the "Insert Candidate" / "Update Candidate" form will be reset.
