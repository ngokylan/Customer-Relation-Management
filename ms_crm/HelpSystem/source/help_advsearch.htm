Advanced Search form

The Advanced Search form extracts and displays Candidate's details from the CRM database and ISYS application, in a user-friendly search functionality.

The "Candidate name" field allows to search a Candidate by First and/or Last Name, using an auto suggest user interface feature that extend search functionality and make easier and fast for Agent to search structured Candidate data.
However, the "Advanced Search" section consent to run an higher complex research based on Skill Set and/or Status and/or Next Available contact date and/or State and/or Keyword by clicking the "Search" button.
Both the searches are independent, however, the displayed result list is determinate based on which button the Agent clicks "GO" or "Search", regardless the parameters entered in other fields.



The displayed data is sorted by hit count (Percentage of matching criteria) and Candidate name, and you can get a quick preview of the Candidate detail by moving the mouse over the candidate name on the retrieved list.

Searching Criteria

Typing on the "Candidate name" field by:

  • First Name
  • Last Name

Entering and selecting "Advanced Search" criteria by:

  • Skill Set: Candidates set of qualifications and/or area of expertise and experience
  • Status: Candidates code which identify the current working situation.
  • Next Available contact date: Scheduled data for next contact with the candidate.
  • State: State where the candidate is living or seeking a job.
  • Keyword: Search for any key words.

Retrieved Candidate Name List

The search is invoked by clicking "GO" button and, if the research is successful, a list of Name, Skill Set qualification, Next Available contact date, phone number of Candidates will be displayed on the "Advanced Search" bottom section.

Go, Search & Reset Button

Clicking the "Search" or "GO" button the "Advanced Search" form will be submitted to the server, on the other hand clicking the "Reset" button all the fields of the"Advanced Search" form will be reset.
